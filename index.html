How we will benefit from KISS ?

  • more flexible code
  • easier to extend, modify or refactor
  • other devevlopers can easily understand this code

How we can apply the KISS principle to coding ?
(some tips)

  • Keep your methods small (30-40 lines per method)
  • Ensure your methods has a single responsibility
  • Delete instances, methods or redundant processes within the code base (that are not in use)

You should try to reduce the complexity while maintaining an efficient system.
